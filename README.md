# Sales Dashboard – Power BI

## Project Overview
This project contains a sales analytics dashboard built using Power BI.
The dashboard helps analyze overall sales performance and trends using interactive visuals.

## Data Source
- Excel file containing sales data
- Data cleaned and transformed using Power Query

## Dashboard Features
- Total Sales and Revenue KPIs
- Monthly sales trends
- Product and region-wise performance
- Interactive slicers for analysis

## Tools Used
- Power BI (DAX, Power Query)
- Microsoft Excel
- GitHub (project documentation)

## Repository Structure
- data/raw – Original Excel data
- data/processed – Cleaned data (if applicable)
- powerbi – Power BI dashboard (.pbix file)
